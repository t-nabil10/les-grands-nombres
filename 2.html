2 Décomposition du nombre :

a) Propriété :

On utilise un TABLEAU DE NUMÉRATION pour connaître la valeur de chaque chiffre.

Exemple : je place 803 650.

Lecture du tableau :

  • Le chiffre 8 représente : 8 × 100 000 = 800 000
  • Le chiffre 0 représente : 0 × 10 000 = 0
  • Le chiffre 3 représente : 3 × 1 000 = 3 000
  • Le chiffre 6 représente : 6 × 100 = 600
  • Le chiffre 5 représente : 5 × 10 = 50
  • Le chiffre 0 représente : 0 × 1 = 0

Total : 800 000 + 0 + 3 000 + 600 + 50 + 0 = 803 650
